# How a c program works

```c

source code (.c)

            |
            |
            |
            |
            

1. Preprocessor

preprocessor directive replaced by its value
comment and white spaces removed
expended source code

 
            |
            |
            |
            |
            

2. Compiler

generate assembly code using expanded source code, (source code to target code)

            |
            |
            |
            |
3. Assembler

take assemble code and generate object code.
object code (.obj or .o)

            |
            |
            |
            |
4. Linker

Libraries

|
|
..

.exe (executable file) or .out (linux)


```
